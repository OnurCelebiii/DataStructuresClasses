## Compile
* ```git submodule update --init --recursive``` get gtest library
* ```mkdir build```
* ```cd build```
* ```cmake ..```
* ```make all```

## Run Tests
* ```cd /build/tst```
* ```./LinkedList_tst```

## Run main
* ```cd /build/src```
* ```./LinkedList_run```
